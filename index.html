Últimos Projetos

Programa Estacionamento

O projeto de estacionamento em React permite o registro eficiente de veículos, gerenciando entradas e saídas de maneira intuitiva. Inclui funcionalidades para visualização de veículos estacionados, cálculo de tempo de permanência e cobrança automática. A interface é amigável e responsiva, facilitando o uso tanto em desktops quanto em dispositivos móveis. Além disso, oferece relatórios detalhados para uma gestão completa do estacionamento.

Parallax da Maria

O projeto "Parallax da Maria" destaca a Festa do Divino 2022, oferecendo uma visão detalhada do evento com uma interface atrativa. Apresenta informações sobre a celebração, incluindo fotos, cronograma e detalhes históricos. O design é responsivo e intuitivo, proporcionando uma navegação agradável tanto em dispositivos móveis quanto em desktops. Além disso, o site capta a essência cultural da festa, engajando os visitantes com conteúdo rico e visualmente encantador.

Lista de Tarefas

O projeto "Lista de Tarefas" é uma aplicação web desenvolvida com React, focada em gerenciar tarefas diárias de forma eficiente. As funcionalidades incluem a adição, edição e exclusão de tarefas, além de marcar tarefas como concluídas. A interface é intuitiva e responsiva, proporcionando uma experiência de usuário fluida em diversos dispositivos. O projeto utiliza tecnologias modernas como React para a interface, CSS para estilização e armazenamento local para persistência de dados, garantindo uma aplicação rápida e funcional.

Spotify

O projeto "Front do Spotify" é uma aplicação web inspirada na interface do Spotify, desenvolvida utilizando Next.js e React. As funcionalidades incluem a navegação por playlists, descoberta de novas músicas e acesso a rádios temáticas. A interface é intuitiva e moderna, oferecendo uma experiência de usuário similar à do Spotify oficial. Tecnologias como Next.js para renderização do lado do servidor e React para construção de componentes dinâmicos garantem uma aplicação rápida e eficiente.

Corrosel de Fotos

O projeto "Carrossel de Fotos" é uma aplicação web que exibe um carrossel interativo de imagens. Desenvolvido com HTML, CSS e JavaScript, permite a navegação suave entre várias fotos, oferecendo uma experiência visual atraente. A interface é responsiva, adaptando-se bem a diferentes tamanhos de tela, o que torna a visualização eficiente tanto em dispositivos móveis quanto em desktops. A simplicidade e a fluidez das transições garantem uma experiência de usuário agradável e intuitiva.

Tabela de Livros

O projeto Tabela de Livros oferece uma plataforma intuitiva para gerenciar acervos literários. Com funcionalidades de adição, edição e exclusão de livros, além de categorização eficiente, proporciona uma experiência completa para organização editorial. Com uma interface limpa e responsiva, facilita o controle e a manutenção de coleções de forma eficaz.
